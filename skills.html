It includes data preprocessing, EDA, model training and evaluation, feature importance analysis, and a Gradio web app for real-time predictions with personalized recommendations.

Movie Recommendation System is a Python-based content recommendation project that uses TF-IDF and cosine similarity to suggest movies similar to a given title. It includes EDA, precomputed similarity matrices, reusable artifacts, and a simple query interface for fast recommendations.
